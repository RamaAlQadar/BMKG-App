# Aplikasi Informasi Gempa BMKG
Oleh **Muhammad Al Qadar Ramadhan**  
**NIM: 237200003**

Aplikasi Android sederhana untuk menampilkan informasi gempa terkini dari BMKG.

## Fitur
- Menampilkan gempa terbaru
- Menampilkan daftar gempa M â‰¥ 5.0
- Detail gempa lengkap (tanggal, jam, wilayah, kedalaman, magnitudo, koordinat)
- Menampilkan ShakeMap jika tersedia dari BMKG
- Tombol lihat lokasi di Google Maps

## Teknologi
- Java (Android)
- Volley (HTTP Request)
- Picasso (Load gambar)
- Material Design
